Репозиторий для документирования опыта самостоятельного производства — бытового, цифрового, материального.
Здесь собираются заметки о том, как делать вещи своими руками:

программы и цифровые инструменты
музыка, звук, визуальные материалы
еда и кулинарные эксперименты
простые устройства и механизмы
методы автономного существования

Это не коммерческий проект. Не образовательная платформа. 
Это рабочий журнал практик для тех, кто предпочитает делать, а не покупать.
Фокус — на независимости через умение создавать. Цифровые инструменты здесь не вспомогательные, а базовые: с них начинается самообеспечение в современных условиях.
Всё описанное — для людей.
